# Assignia — система распределения задач по ролям и командам

Assignia (от англ. *assign* — назначать) — веб-приложение для управления задачами 
с гибкой ролевой моделью, публичными и приватными проектами, учётом рисков и аналитикой.

## Основные возможности
- Распределение задач по направлениям и командам
- Назначение исполнителей с учётом ролей
- Публичные проекты и обработка внешних запросов через техподдержку
- Приоритезация задач с оценкой рисков как вероятность и последствия
- Статистика и аналитика по исполнителям и направлениям
